# test-dynamodb

A Clojure project designed to test dynamo db

## Usage

Primary objective was to use dynamo db as atomic counter
no tests yet


## steps
* install docker 
* run `docker run -p 8000:8000 amazon/dynamodb-local`
* run function in core
